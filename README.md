# 函数萃取

## 实现功能

1. 萃取函数返回类型
2. 萃取函数参数类型
3. 构造函数类型
4. 构造函数指针类型
5. 构造 STL.Function 类型
6. 将 lambda 转换为 STL.Function
7. 将 lambda 转换为函数指针

## 支持的类型

1. 普通函数
2. 函数指针
3. STL.Function
4. 类成员函数指针， 支持 const， volatile， volatile const 修饰
5. 可调用对象，如伪函数 (函数对象)，lambda

## 使用的技术

1. 类型萃取
2. C++11/14

# Span

## 实现功能

1. 胖指针
2. 主要是对数组的封装， 可以作为函数参数的封装，以及数组类型的视图构建

## 支持的类型

1. C 风格数组
2. 具有 data() & size() 成员函数的类对象

## 使用的技术

1. 模板元编程 - SFINAE
2. C++11/14
